智慧农业的介绍

        智慧农业就是将物联网技术运用到传统农业中去，运用传感器和软件通过移动平台或者电脑平台对农业生产进行控制，使传统农业更具有“智慧”。除了精准感知、控制与决策管理外，从广泛意义上讲，智慧农业还包括农业电子商务、食品溯源防伪、农业休闲旅游、农业信息服务等方面的内容。
        所谓“智慧农业”就是充分应用现代信息技术成果，集成应用计算机与网络技术、物联网技术、音视频技术、3S技术、无线通信技术及专家智慧与知识，实现农业可视化远程诊断、远程控制、灾变预警等智能管理。
        智慧农业是农业生产的高级阶段，是集新兴的互联网、移动互联网、云计算和物联网技术为一体，依托部署在农业生产现场的各种传感节点（环境温湿度、土壤水分、二氧化碳、图像等）和无线通信网络实现农业生产环境的智能感知、智能预警、智能决策、智能分析、专家在线指导，为农业生产提供精准化种植、可视化管理、智能化决策。
        “智慧农业”是云计算、传感网、3S等多种信息技术在农业中综合、全面的应用，实现更完备的信息化基础支撑、更透彻的农业信息感知、更集中的数据资源、更广泛的互联互通、更深入的智能控制、更贴心的公众服务。“智慧农业”与现代生物技术、种植技术等高新技术融合于一体，对建设世界水平农业具有重要意义。

产品硬件方向介绍

        该作品是集精量种肥同施、滴灌带浅埋铺设、覆膜及水肥一体化两级、高效、精量的智能滴灌控制系统于一体的智能化、精准化、高效化的农业系统装备，主要用于玉米整个生产周期。
        该装备首部包括浅埋滴灌铺管系统、勺式精量播种系统、种肥定量定位输送系统、地膜覆盖与覆土系统、智能检测控制系统。浅埋滴灌铺管系统与自动剪带相结合，使滴灌带张紧进行高稳定的铺设。勺式精量播种系统与智能检测同步补种相结合，使缺种率降到最低。种肥定量定位输送系统,槽轮安装在肥箱底部，通过旋转将种肥刮入落肥管来控制施肥量，送至在圆周方向设有窝眼的窝眼轮来定位施肥。地膜覆盖与覆土系统，锥形夹持装置将地膜辊装夹，震土辊压与地膜之上，保证地膜的平整，后跟覆土，对地膜两侧进行覆土掩盖。
        该装备尾部包括整个滴灌带网络，分区肥料混合站，园区主控制站，分区无线控制平台，无线基站检测装置，肥箱检测装置，各种控制阀，总站蓄水池及净化装置，增压泵、流量阀、压力计等。园区总站启动后，可以通过手动或导入数据库函数，实现对某个区域作物需要检测数据上下限值、报警值的预先设定，同时无线检测基站的EC传感器开始工作，将动态数据传送给总站，至此完成正常情况下的检测反馈。当检测值处于设定值范围內时，分区无线控制平台持续休眠，当检测值低于下限值时，分区无线控制平台激活，开始第一次纯水滴灌，当达到一定湿度时，无线检测基站的氮、磷、钾检测传感器开始工作，将测到的值传送给园区主站，然后主站控制分区无线控制平台、分区肥料混合站、肥箱检测装置等同步工作，混合成所需浓度的营养液，至此肥料配比结束。肥箱检测装置发出配肥结束信号，分区无线控制平台在主站控制指令的指导下，开始第二次营养液滴灌，同时无线检测基站的氮、磷、钾检测传感器会将数据动态反馈给园区总站，逐一关闭各分区各支路滴灌装置，最终完成滴灌。此外该作品分区无线控制平台和无线基站检测装置采用太阳能供电，并利用蓄电池储能，可实现连续性工作。
        该装备通过六大系统的配合，在实现高效播种施肥灌溉覆膜的同时，也提高了机器适应性和农业耕种的工作效率，又能实现对农作物整个生产周期内的精准营养供给，有效的解决农业缺水、肥料利用率低、产出率低的问题，实现绿色、高效的现代化农业。

产品软件方向介绍

        本系统就是针对土壤元素检测识别而开发的，旨在辅助农耕人员通过本系统完成对土壤元素检测的需求。从而节省更多的时间和经历投入到其他的工作之中。本系统的目的在于分析土壤的含水盐值以及有用的氮（N）、磷（P）、钾（K）元素的含量 , 将它们在页面上展示出来, 并且通过本系统，日常对土壤的检测，维护，查验工作进行管理。摒弃了传统的手工纸质记录的方式，大大提升了工作的效率，减少了可能出现的认为错误。
        前端使用Bootstrap ，是一个基于HTML、CSS、JavaScript 的开源框架。该框架代码简洁、视觉优美，可用于快速、简单地构建基于 PC 及移动端设备的 Web 页面需求。 是内置很多 JavaScript 插件和图标的多功能 Web 前端的开源框架。 其最为重要的部分就是它的响应式布局，通过这种布局可以兼容 PC 端、PAD以及手机移动端的页面访问。jQuery是一个快速、简洁的JavaScript框架，是继Prototype之后又一个优秀的JavaScript代码库（或JavaScript框架）。和jQuery，其封装JavaScript常用的功能代码，提供一种简便的JavaScript设计模式，优化HTML文档操作、事件处理、动画设计和Ajax交互。
        后端使用Java web的SSM框架，是Spring + Spring MVC + MyBatis的缩写，这个是继SSH之后，目前比较主流的Java EE企业级框架，适用于搭建各种大型的企业级应用系统。
        数据库使用MySQL，由于性能高、成本低、可靠性好，已经成为最流行的开源数据库，因此被广泛地应用在Internet上的中小型网站中，是最流行的关系型数据库管理系统。
